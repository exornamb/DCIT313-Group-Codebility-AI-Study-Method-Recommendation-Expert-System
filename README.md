## **System Objective**

The Study Method Recommendation Expert System is a rule-based artificial intelligence system that provides students with appropriate study methods based on their own study habits and learning styles. It collects user data on their own study behavior and uses a Prolog-based knowledge base containing logical rules to recommend appropriate study methods. The recommended study methods include **Active Recall, Spaced Repetition, Mind Mapping, Pomodoro Technique, Group Study, and Practice-Based Learning.**
